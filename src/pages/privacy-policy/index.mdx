This **Privacy Policy** applies to the [dav.one](https://dav.one/) website and its preview version available at [next.dav.one](https://next.dav.one/).
**Last Updated:** June 26, 2026

## Overview

This website is a [personal programming project](https://dav.one/about-website/).  
Privacy is respected through minimal data collection practices and cookie-free operation.

## Data Collection

### Technical (HTTP) data

The site is hosted on [Cloudflare Pages](https://pages.cloudflare.com/), 
and **Cloudflare's standard HTTP traffic logging** may record typical request/response metadata.
This can include:
- IP address (and derived approximate location, e.g., country/region)
- Date and time of the request
- Requested URL path and query string
- HTTP method (GET/POST/etc.)
- Response status code
- Referrer header (if provided by the browser)
- User-Agent header (browser/device information)

### Analytics

This website uses two privacy-focused analytics solutions:

**[Cloudflare Web Analytics](https://www.cloudflare.com/web-analytics/)** is enabled to collect basic usage metrics. It is a privacy-first tool that does not use client-side state, such as cookies or localStorage, to collect usage metrics, and it does not track users across sites.

**[PostHog Web Analytics](https://posthog.com/web-analytics)** is configured with [in-memory persistence only](https://posthog.com/docs/libraries/js/persistence), meaning no cookies or persistent data are stored in your browser.
Data collected is limited to basic usage patterns and is not tied to persistent identifiers.

## Cookies

This website **does not use** [cookies](https://en.wikipedia.org/wiki/HTTP_cookie). Both of the analytics tools mentioned above operate entirely without cookies.

## Data Processing

HTTP request data and basic web analytics are processed by Cloudflare as part of the hosting and analytics infrastructure.
For information about your privacy rights and how Cloudflare handles data, see [Cloudflare's Privacy Policy](https://www.cloudflare.com/privacypolicy/).

## Data Retention

**Cloudflare Data:** HTTP logs and basic analytics metrics are retained temporarily by Cloudflare for security, infrastructure, and analytical purposes.
Refer to [Cloudflare's Privacy Policy](https://www.cloudflare.com/privacypolicy/) for details.

**PostHog Analytics:** With in-memory persistence, data exists only during your active session and is automatically discarded when you close the tab or navigate away.
No long-term retention occurs.
