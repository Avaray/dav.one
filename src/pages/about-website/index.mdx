The website you're looking at is the result of my desire to share knowledge on topics that interest me, as well as my passion for exploring libraries for creating modern web applications. 

To be able to share an impressive total of {Object.keys(articles).length} articles with the world, I used tools and technologies such as:  
- [Astro](https://astro.build/) - a modern static site generator.
- [TypeScript](https://www.typescriptlang.org/) - a typed superset of JavaScript.
- [TailwindCSS](https://tailwindcss.com/) - a utility-first CSS framework.
- [UnoCSS](https://unocss.dev/) - an instant on-demand atomic CSS engine.
- [daisyUI](https://daisyui.com/) - awesome library of components for TailwindCSS.
- [Iconify](https://iconify.design/) - a huge library of icons.
